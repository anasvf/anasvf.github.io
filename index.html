Weather Station Using Arduino

GITHUB

A Weather station is a device that collects data related to the weather and enviroment using many different sensors. We can measure many things like: Temperature Humidity Wind Barometric Pressure UV index Rain In the Weather Station that we are going to build we are going to measure Temperature and Humidity in two locations and display the current date and time. Building a weather station is extremely easy. But can a maker build a unit with a color TFT display and features that match those of a commercial unit? The answer is YES! With the power of open source software and hardware, one can build this impressive weather station easily! The project consists of two parts, the transmitter and the receiver. The transmitter measures the temperatutre and the humidity and send the data wirelessly to the receiver. The receiver, measures the temperatutre and the humidity, receives the data from the remote sensor and displays everyting in big color tft display.
